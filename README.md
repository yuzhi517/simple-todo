# Simple Todo

> 一个命令行的待办事项管理工具 | 可以完整支持中文 | 双击就能用

这个工具是一个比较简单的 CLI 任务管理器，主要是为了中文用户的使用来设计的，中英文混合输入和输出都可以正常处理，在终端里显示的表格也能够对齐。使用的时候不需要去记什么命令，直接双击启动文件就行了（macOS 双击 `todo.command`，Windows 双击 `todo.bat`），会弹出一个菜单界面让你操作。

## 功能概览

- ✅ **添加 / 删除 / 完成任务** — 基本的增删改操作都包含了
- ✅ **中文的显示处理** — 标题可以写中文的，终端里的对齐不会出现偏移的问题
- ✅ **优先级管理** — 可以用数字给任务标重要程度，数字越小表示越重要，排得越靠前，有独立的列来显示
- ✅ **搜索功能** — 可以通过输入关键词来快速找到任务
- ✅ **没有外部依赖** — 只用到了 Python 自带的那些标准库
- ✅ **JSON 存储** — 数据文件是 JSON 格式的，可以直接打开来看和修改

## 怎么开始用

### 环境要求

- 需要有 Python 3.8 或者更高的版本
- macOS、Linux 或者 Windows 系统都可以用

### 使用方法一：双击运行（推荐）

在文件夹里面找到启动文件，用鼠标双击它就行了：
- **macOS** → 双击 `todo.command`
- **Windows** → 双击 `todo.bat`

系统会弹出一个终端窗口，里面有菜单可以选：

```
========================================
         Simple Todo — 待办事项
========================================

  （当前的任务列表会显示在这里）

----------------------------------------
  1. 添加任务
  2. 完成任务
  3. 恢复任务（取消完成）
  4. 删除任务
  5. 搜索任务
  6. 修改优先级
  7. 查看全部（含已完成）
  8. 刷新列表
  0. 退出
----------------------------------------

请选择操作 [0-8]:
```

然后按照提示去选数字就行，不需要记任何命令，想要用哪个功能就按哪个数字。

### 使用方法二：在终端里手动敲命令

如果不方便用双击的方式，也可以打开终端，进到项目目录里面手动敲命令来用。

```bash
cd simple-todo

# 往列表里添加任务
./todo.py add "完成中期报告"
./todo.py add "交作业" -p 1

# 查看任务的列表（这个是不写命令时的默认行为）
./todo.py

# 把任务标记成已完成
./todo.py done 1

# 搜索任务
./todo.py search 报告
```

## 命令参考

如果不使用菜单的话，也可以直接在终端里用下面的命令：

| 命令 | 说明 | 示例 |
|------|------|------|
| `add <标题>` | 往列表里加入一条新的任务 | `todo add "买菜" -p 1` |
| `list` | 把还没完成的任务列出来（这个是默认的） | `todo` 或者 `todo list` |
| `all` | 把所有的任务都列出来，包含已完成的 | `todo all` |
| `done <ID>` | 把指定编号的任务改成已完成的状态 | `todo done 1` |
| `undone <ID>` | 把已完成的任务恢复成未完成 | `todo undone 1` |
| `rm <ID>` | 把任务从列表里删掉 | `todo rm 1` |
| `search <关键词>` | 在标题里搜索包含某个词的任务 | `todo search 报告` |
| `priority <ID> <数字>` | 修改某条任务的优先级 | `todo priority 1 1` |
| `help` | 把帮助信息显示出来 | `todo help` |

### `add` 命令的选项

| 选项 | 说明 |
|------|------|
| `-p, --priority <数字>` | 给任务设置优先级（不写的话默认是 1，数字越小就排在越前面） |

## 排序规则

任务在被列出来的时候会按照下面的顺序来排列：

1. **完成状态** — 还没有完成的任务会被放在前面来显示
2. **任务的优先级** — 数字越小的任务会被排在越靠前的位置（1 是最重要的，2 次之，3 再次，从小到大排列）
3. **创建的时间** — 优先级一样的时候，创建得更早的任务会排在前面（从早到晚排列）

## 数据存储

程序把所有的数据都存放在 `~/.simple_todo/tasks.json` 这个文件里面，数据的格式像下面这样：

```json
[
  {
    "id": 1,
    "title": "完成中期报告",
    "priority": 1,
    "done": true,
    "created_at": 1752053425.123,
    "done_at": 1752053520.456
  }
]
```

- **数据可以手动去修改** — 随便用一个文本编辑器打开这个文件就能直接改里面的内容
- **备份起来也很简单** — 只要把 `~/.simple_todo/` 这个目录整个复制一份就行了

## 中文支持说明

为了让中文能够正常使用，这个工具在下面几个方面做了处理：

| 方面 | 用的方法 |
|------|----------|
| **内容的存储** | JSON 文件编码用的是 UTF-8，写文件的时候设置了 `ensure_ascii=False` |
| **终端的显示** | 调用 `unicodedata.east_asian_width()` 这个函数来检查每个字符的宽度 |
| **表格的对齐** | 中文的字符会按照两个列的宽度来计算，英文按照一个列的宽度来算，优先级也有单独的一列 |

不管是纯中文的标题，还是中英文混在一起的，甚至包含了一些 emoji 的情况，表格显示出来都不会歪。

## 设计原则

1. **尽量简单** — 整个程序只用一个 Python 文件来实现，不依赖任何外部的第三方库
2. **双击就能用** — 不需要先打开终端再敲命令去启动，直接双击对应的启动文件（macOS 用 `.command`，Windows 用 `.bat`）就会弹出交互菜单
3. **优先考虑中文** — 从终端显示的表格对齐到各种错误提示的文字，再到菜单的引导，都做了中文的原生支持
4. **数据是透明的** — 用的是 JSON 格式来存数据，用户可以随时打开文件去查看和修改
